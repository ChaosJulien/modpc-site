BigDick 转服工具

ℹ️ 本页面介绍 BigDick 项目，一个用于 MODPC 社区“布吉岛”服务器转服的开源工具。

📌 项目概述

BigDick 是一个基于 .NET 的开源转服工具，主要用于：

  • 实现从 MODPC 主服切换至“布吉岛”服务器（bjd转服）
  • 绕过联机大厅的锁服限制，自由加入指定服务器

项目采用 GPL-3.0 开源协议，源码公开，允许学习与修改。

📦 源码与构建

项目仓库地址：

GitHub: ChaosJulien/BigDick

使用步骤：

  1. 克隆或下载项目源码
  2. 使用 Visual Studio 或 .NET CLI 构建 Mcl.Core.dll
  3. 将生成的 DLL 文件替换到你的 MCLauncher 安装目录下的对应位置
  4. 重启启动器即可生效
💡 建议：如果你不熟悉 .NET 构建流程，可尝试在社区寻找他人编译好的版本（注意安全验证）。

🔧 主要功能

  • 支持 bjd 转服，快速跳转至布吉岛服务器
  • 实现 联机大厅反锁服，避免被勒索无法进入联机大厅服务器
  • 轻量集成，仅需替换核心 DLL 文件

❓ 常见问题

必须自己构建吗？

是的，该项目未提供预编译版本，需开发者自行编译 Mcl.Core.dll。

有风险吗？

修改启动器核心文件可能触发反作弊机制或导致崩溃，建议备份原文件。

是否有替代方案？

目前部分新版启动器已内置转服功能,如null挂福器，可优先尝试使用集成化工具，降低操作门槛。
